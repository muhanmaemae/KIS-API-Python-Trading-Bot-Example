✨ 주요 기술적 특징 (Key Features)
💎 V-REV 역추세 하이브리드 엔진 : 0주 새출발 시 상승장 소외(FOMO)를 막는 1층(1.15배 할증)과 단기 휩소 하락을 노리는 2층(0.975배 하향)으로 매수 타점을 50%씩 디커플링합니다. 수수료 절대 방어를 위해 1층 앵커는 전일 종가 기준 1.006배로 강제 고정하여 단기 현금 창출력을 극대화합니다.
💎 잭팟 & 1층 전량 강제 청산 (Sweep Finisher) : 장 마감 2분 전(15:58 EST), 잭팟 커트라인(1.010배) 돌파 시 잔량을 덤핑하며, 주가가 1층 타점 이상을 유지 중이면 1층 잔여 물량까지 시장 매수호가로 100% 폭격하여 지층을 완벽히 소각합니다.
💎 애프터마켓 3% 로터리 덫 (Lottery Trap) : 정규장 마감 직후(16:05 EST / 05:05 KST), 유동성 공백과 실적 발표 노이즈를 활용하여 잔여 물량 100%를 총 평단가 대비 +3% 타점에 장후 지정가(AFTER_LIMIT)로 자동 셋팅하여 극단적 꼬리 수익을 포획합니다.
💎 삼위일체 소각 엔진 & 마이너스 손절 개방 : 텔레그램 수동 리셋 격발 시 본장부, 가상장부(Escrow), LIFO 큐(Queue), 숨겨진 백업본까지 4중 찌꺼기 데이터를 100% 동시 소각합니다. KIS 실잔고 0주 스캔 시 누적 수익이 마이너스라도 즉각 장부를 비우고 새출발 모드로 락다운을 강제 해제합니다.
💎 야후 파이낸스 데드락 방어 : 외부 데이터 연산 스레드 무한 대기(Hanging)로 인한 시스템 교착 및 텔레그램 무한 스팸 버그를 차단하기 위해 모든 yfinance 호출에 timeout=5 파라미터를 강제 주입하고 KIS API 우회 로직으로 즉각 전환합니다.
💎 스마트 장부 동기화 엔진 (TrueSync) : 미국장 영업일 스케줄을 자동으로 인식하며, 과거 데이터를 덮어쓰지 않고 오차만큼만 교정하는 비파괴 보정(CALIB) 및 야후 연동 액면분할 자동 소급 로직을 지원합니다.
🛠️ 설치 및 실행 방법 (Installation & Usage)

✔️ Python 3.12 이상
✔️ 한국투자증권 Open API 발급 (App Key, App Secret)
✔️ Telegram Bot Token 및 Chat ID

프로젝트 최상단 폴더에 .env 파일을 만들고 아래 양식에 맞게 본인의 키를 입력합니다.
TELEGRAM_TOKEN=나의_텔레그램_봇_토큰

APP_KEY=나의_한국투자증권_APP_KEY
APP_SECRET=나의_한국투자증권_APP_SECRET
CANO=나의_계좌번호_앞8자리
ACNT_PRDT_CD=01 또는 22
📌 4. 프로그램 실행
python main.py
(권장: 서버 환경에서는 nohup python main.py & 명령어를 사용하여 백그라운드에서 24시간 가동되도록 설정하세요.)
📂 파일 구조 (Directory Structure)
📁 main.py: 스케줄러 구동, 의존성 주입(DI) 라우팅 및 프로그램의 메인 진입점.
📁 scheduler_core.py: API 토큰 갱신, 장부 동기화, 시스템 자정 작업 등 생명 유지를 전담하는 백그라운드 관리 코어.
📁 scheduler_trade.py: 정규장 LOC 장전, 스나이퍼 감시, VWAP 타임 슬라이싱 및 스윕 피니셔, 애프터마켓 로터리 덫을 전담하는 매매 스케줄러.
📁 strategy_reversion.py: V-REV 역추세 하이브리드 타점 산출 및 공수 스위칭 엔진.
📁 queue_ledger.py: V-REV 날짜 기반 지층 통합 LIFO 큐 데이터 비파괴 관리 모듈.
📁 vwap_strategy.py: 장 마감 30분 전 자율주행 유동성 분할 및 1호가 정밀 타격 엔진.
📁 volatility_engine.py: 1년 ATR 평균 및 VXN/HV 기반 시장 변동성 마스터 스위치 연산 모듈.
📁 broker.py: 한국투자증권 API 통신 및 야후 파이낸스 데이터(데드락 방어 적용) 가공 클래스.
📁 strategy.py: 일반 무매(V14/V17) 예산 분배 및 목표가 산출 알고리즘 클래스.
📁 telegram_bot.py / telegram_view.py: 텔레그램 봇 컨트롤러(삼위일체 소각 엔진 탑재) 및 통합 지시서 대시보드 화면 렌더링 로직.
📁 config.py: 각종 JSON 데이터를 저장하고 불러오는 로컬 캐싱 및 설정 매니저.
📁 version_history.py: 코드 업데이트 최신 히스토리 기록.
